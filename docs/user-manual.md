# Manual corto de usuario

## Login

1. Inicia sesion con tu correo y contrasena.
2. Si el acceso falla, valida que el usuario siga activo.

## Registrar productos

1. Entra al modulo `Productos`.
2. Completa codigo, nombre, precios y stock.
3. Guarda el producto.

## Registrar insumos

1. Entra al modulo `Insumos`.
2. Registra nombre, costo, stock y vencimiento si aplica.
3. Guarda el insumo.

## Registrar ventas

1. Entra al modulo `Ventas`.
2. Selecciona productos y cantidades.
3. Revisa el total.
4. Confirma la venta.

## Registrar produccion

1. Entra al modulo `Produccion`.
2. Selecciona el producto a elaborar.
3. Ingresa cantidad, lote y vencimiento.
4. Guarda la produccion.

## Ajustar inventario

1. Entra al modulo `Inventario`.
2. Elige si el movimiento afecta producto o insumo.
3. Selecciona entrada, salida o merma.
4. Registra cantidad y motivo.

## Revisar reportes

1. Entra al modulo `Dashboard` para ver resumen diario.
2. Entra a `Reportes` para revisar ventas, inventario y produccion por fechas.

## Recomendaciones de uso diario

- registrar ventas en el momento en que ocurren
- registrar produccion el mismo dia de elaboracion
- registrar mermas apenas se detecten
- revisar stock bajo al inicio o cierre de jornada
