# Alcance del MVP

## Objetivo

Construir un sistema web funcional para una reposteria que permita registrar ventas, productos, insumos, produccion e inventario desde el dia 1, dejando preparada la base para habilitar prediccion cuando exista al menos 1 mes de datos reales.

## Incluye en la primera version

- Login de usuarios
- Gestion de productos
- Gestion de insumos
- Registro de ventas
- Control de inventario
- Registro de produccion
- Dashboard basico
- Reportes basicos

## No incluye en la primera version

- Prediccion activa
- Integracion automatica con clima
- Pedidos personalizados complejos
- App movil nativa
- Notificaciones automaticas

## Criterios del MVP

- Debe permitir operacion diaria desde el primer dia
- Debe mantener trazabilidad basica de ventas, produccion e inventario
- Debe usar autenticacion y roles basicos
- Debe dejar datos listos para analisis futuro

## Entregables esperados del MVP

- Backend funcional con API
- Base de datos PostgreSQL con migraciones
- Frontend web operativo
- Dashboard y reportes iniciales
- Documentacion tecnica y manual corto de uso
